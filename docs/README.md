# BMS BLE 项目开发与使用手册

为了保障项目的可维护性、高内聚性与未来的多协议扩展性，项目将蓝牙通信、系统权限及授权保护逻辑进行了高度解耦。本目录包含一系列细分文档，分别指导开发与维护人员理解并扩展系统。

---

## 📖 文档索引 (Documentation Index)

请点击以下链接查看对应模块的具体技术文档：

### 1. [系统整体架构与职责划分](./1.architecture.md)
*   **内容**: 项目的分层设计、核心模块（配置层、物理服务层、协议策略层、状态总线层、业务 Hook 层、工具函数沙盒）的物理目录分工与文件职责。
*   **适用场景**: 首次接触项目、理解全局数据流向与职责边界。

### 2. [蓝牙物理连接生命周期](./2.connection_lifecycle.md)
*   **内容**: 物理连接时序图、停止扫描天线释放（防 Android 10003）、服务与特征值发现（防 iOS/鸿蒙 10004）、UUID 格式大小写对齐、动态 MTU 协商与 GATT 缓存就绪延时机制。
*   **适用场景**: 连接异常调试、物理连接稳定性维护。

### 3. [指令收发与队列调度机制](./3.transceiver_queue.md)

*   **适用场景**: 通信时序冲突、数据丢包、响应超时熔断排查。

### 4. [多协议热兼容与策略模式](./4.multi_protocol.md)

*   **适用场景**: 接入新硬件协议、开发品牌专属首页及控制面板。

### 5. [核心业务页面功能使用指南](./5.page_guide.md)
*   **内容**: 蓝牙设备搜索页（z-paging 本地托管、打字模糊过滤）、电池遥测首页（呼吸灯、SOC、Ah 融合、6路温度过滤）、参数控制页（加热、紧急清除/休眠/启动）、个人中心与设置（多语言切换 reLaunch 重绘、莫兰迪色热重绘、114514 日志解锁）。
*   **适用场景**: UI 功能调试、用户使用手册参考、全局配置与日志解锁。

### 6. [系统安全授权与激活校验](./6.auth_activation.md)
*   **内容**: 九位特征设备识别码 `code_dev` 随机生成、8位激活码离线逆向解码、比对校验校验码、成功后特征码废除重置（防重放重刷）、全局 `isAuthorized` 开关状态还原保护。
*   **适用场景**: 安全机制审查、设备计费与过期锁调试。

### 7. [系统权限诊断与自适应修复](./7.permission_diagnostics.md)

*   **适用场景**: 申请权限被拒、权限跳转卡死修复。

### 8. [常见问题排查与避坑指南](./8.troubleshooting.md)

*   **适用场景**: 视觉动效调试与性能维护。
