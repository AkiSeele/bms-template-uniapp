# 5. 核心业务页面功能使用指南

项目包含了五个主要核心页面和几个系统设置诊断面板，均深度遵循 wot-ui v2 组件设计规范与 UnoCSS 样式类。

---

## 一、 蓝牙设备搜索页 ([ble-search/index.vue](../pages/ble-search/index.vue))

*   **功能亮点**: 本地分页数据托管、下拉刷新极连扫描、打字实时筛选。
*   **交互逻辑**:
    1.  进入页面时通过 `onShow` 自动拉起 reloading 触发诊断。若蓝牙或位置服务未就绪，跳转引导。
    2.  调用 `uni.startBluetoothDevicesDiscovery` 启动限时扫描（默认 8 秒）。
    3.  列表采用 `z-paging` 进行本地托管去重过滤。支持对搜索出的设备名称和物理 MAC 进行打字模糊搜索。
    4.  点击列表项后，自动关闭扫描，触发物理连接时序。

---

## 二、 电池数据首页 ([index/index.vue](../pages/index/index.vue))

*   **功能亮点**: 脉冲呼吸灯连接反馈、电池状态大图标展示、均压差及多传感器数据融合。
*   **交互逻辑**:
    1.  顶部自定义 Navigation Bar，右侧集成扫码一键配对入口，左侧联机状态自适应展示。
    2.  中心动力圆盘环形展示当前剩余 SOC（%）。当充电时图标自适应呈现闪电呼吸动效，并用高阶计算属性显示 `Ah` 的绝对电量。
    3.  核心参数网格实时更新电压、电流、温度和 SOH 健康度。
    4.  温度监控区自动遍历 `temperatures` 数组，**智能过滤未接入传感器的无效预设值（`0` / `0xFF` / `255`）**，仅展示有效的电芯传感器值、MOS 温度和环境温度。
    5.  底部提供充电、放电开关组件以及防连点 Loading 遮罩。

---

## 三、 参数控制页 ([control/index.vue](../pages/control/index.vue))

*   **功能亮点**: 未连接状态引导、危险动作二次弹窗、全局授权锁保护。
*   **交互逻辑**:
    1.  检测 `isBleConnected`：未连接展示动态转动齿轮图并提供回连通道；连接时展示当前品牌及对应协议。
    2.  提供充电、放电控制开关。如果是特定协议设备（如支持加热的 `protocol-a`），自适应呈现“低温自加热”开关。
    3.  紧急维护区（限特定协议可用）提供：清除异常报警、强制板子深度休眠、强制拉起激活工作等硬件指令下发。
    4.  所有控制指令触发前，均先校验 `isAuthorized`。未授权用户强制拦截反转开关，并弹出激活框。
    5.  危险性维护指令在执行前，均有 `dialog.confirm` 二次弹窗提醒。

---

## 四、 个人中心页 ([mine/index.vue](../pages/mine/index.vue) & [settings.vue](../pages/mine/settings.vue))

*   **功能亮点**: 离线/云端联机卡片、多语言 reLaunch 重载、全站热重绘色板、隐藏系统日志解锁。
*   **交互逻辑**:
    1.  **用户中心卡片**: 离线模式下展示离线游客身份，禁用云端服务；联机模式下支持模拟接口登录及注销。
    2.  **多语言切换**: 支持中/繁/英三语。点击 ActionSheet 选项后通过 `appStore.setLocale` 持久化，并在 toast 提示结束后**强制执行 `uni.reLaunch` 重载当前个人中心页面**。此重载可强制刷新非当前页面组件（如自定义底部 Tabbar 及顶部导航栏）的国际化重绘。
    3.  **主题切换**: 使用 Segemented 扁平选择卡片，无缝热切换 `light`、`dark`、`system` 明暗模式。
    4.  **色彩配置 ([settings.vue](../pages/mine/settings.vue))**: 允许手动输入色值或选择预设莫兰迪色芯片热覆盖系统的“主题色”、“警告色”、“成功色”和“危险色”，由 Pinia 自动重绘全站关联图标和按钮。
    5.  **系统日志解锁**: 日志页面单元格默认隐藏。在特定设置或点击行为触发密码弹窗后输入 `114514`，成功后解锁隐藏入口，可直接跳转查看底层的蓝牙连接和通信日志。

## 五、 固件升级页 ([mine/firmware-update.vue](../pages/mine/firmware-update.vue))

*   **功能亮点**: 跨平台文件选择自适应、物理防手势返回拦截锁、升级进度段状态流转。
*   **交互逻辑**:
    1.  **分平台文件选择**:
        *   **微信小程序**: 反射调用微信官方 `wx.chooseMessageFile`，允许用户直接从微信会话列表中挑选最新的 `.bin` 或 `.hex` 固件文件。
        *   **安卓原生 App**: 动态请求外部存储卡读写权限，调起原生文件选择插件 `lemonjk-FileSelect` 自定义挑选本地存储器中的文件。若无权限则弹出强对话框引导至系统应用设置。
    2.  **安全写入锁与拦截机制**:
        *   一旦点击“开始升级”进入更新中状态，Pinia 状态仓的 `isOtaUpdating` 置为 `true`。
        *   指令调度队列立刻清除全部正在排队的 `poll` 普通遥测心跳并将其锁定挂起，确保固件擦写大包文件独占蓝牙通信信道。
        *   利用 `onBackPress` 拦截 App 端的用户物理返回按键及侧滑返回手势，并且在小程序端调用 `wx.enableAlertBeforeUnload`，防范用户在更新中途强退导致电池 BMS 芯片损坏变砖。
    3.  **步骤状态机与重置**:
        *   界面伴随“选择固件 -> 校验格式 -> 物理擦写 -> 更新完成”的四级 Steps 进度跃迁。
        *   升级期间由 GSAP 补间动画在 Vue 层面驱动上传圆圈浮动、外环脉冲扩散以及指示灯闪烁。
        *   更新成功后提供重置按钮，恢复常规轮询并安全解锁通道。
